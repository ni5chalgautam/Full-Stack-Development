This portfolio demonstrates my understanding of Full Stack Development.

Full stack development involves working on both front-end and back-end technologies. A full stack developer must be proficient in client-side (HTML, CSS, JavaScript) and server-side (Node.js, Python, PHP) development, along with databases (MySQL, MongoDB). Full stack developers can build complete web applications from scratch.

HTML (HyperText Markup Language) is the foundation of web pages. It defines the structure of a webpage using elements like headings, paragraphs, images, and links.

<h1>Hello, World!</h1>

HTML is essential for defining content and ensuring accessibility.

CSS (Cascading Style Sheets) is used to style web pages. It allows developers to control layout, colors, fonts, and responsiveness.

body { background-color: lightblue; color: darkblue; }

CSS frameworks like Bootstrap make styling easier and more consistent.

Bootstrap is a popular front-end framework that provides ready-made components like grids, buttons, and modals for responsive web design.

You clicked the Bootstrap button!

Using Bootstrap simplifies development and ensures consistency across devices.

JavaScript allows for interactive web pages. It enables dynamic content updates, event handling, and animations.

JavaScript frameworks like React and Vue.js enhance front-end development.
